What's New in InterMapper®?

InterMapper includes many new features and enhancements. Here's a summary...

Web server identification

InterMapper identifies an HTTP server during auto-discovery, and creates a probe groups automatically to monitor it.

ARP discovery

InterMapper uses the ARP (Address Resolution Protocol) table to find additional devices during auto-discovery, broadening its view of the network.

UI Enhancements

Many parts of the user interface have been modernized. A number of menu items have been consolidated to improve usability.

  • The Info Window (Pg 1) has been expanded and redesigned to include many of the Set Info menu commands, and will eventually replace them.
  • A new Welcome page (Pg 1) to helps you get started more quickly.

InterMapper Reports

InterMapper Reports now works on the iPad and other modern web-capable mobile devices.

  • Datasets are better organized, and each user-created dataset is labeled with its probe and variable name.
  • Many improvements have been made to charts, including a cleaner visual design, smarter axis scaling and value selection, and a legend.
  • A number of improvements result in faster load times, better interface responsiveness, and overall performance.
  • Units of measurement are now shown throughout Reports, for any dataset that specifies them.
  • The Adobe Flash plug-in is no longer required to display charts.

Data Management

Additional tools have been added to let you manage how data is exported to the database.

  • The probe <autorecord> section is deprecated in favor of a new <datasets> section, which can be used for any dataset.
  • The new <datasets> section lets probe authors specify the units in which the data will be shown.
  • Probe, variable and units columns have been added to the dataset table, making it easier to write SQL queries against user-created datasets.
  • The Database overview page content has been condensed into a more useful form.

New fields in labels, notifiers and status windows

  • Several Entity MIB fields are now available when editing a device label.
  • The sysName, sysDescr, MAC Address, and several Entity MIB fields are available when editing a notifier message.
  • Several Entity MIB fields are displayed in the status window for an SNMP probe.

Minor Features

  • Sounds are turned off by default for new installs, but are easily turned on .
  • Server Settings, Map Settings and Preferences windows on Windows and Linux platforms contain a number of visual improvements.
  • Scaling of Reports charts has been improved so that a chart fits the page when printing.
  • You can now drag-and-drop Label variables to add them to labels.
  • In the Send Feedback window, you can now update an open ticket by providing the ticket number.
  • A number of bug fixes and minor improvements have been made.
