Server Preference Panels

Use the following panels of the Server Settings window to set global preferences for the selected server.

 

In the left pane of the Server Settings window, click a button to view and edit its settings, as follows:

  • SNMP
  • Log Files
  • DNS/WINS settings
  • E-Mail
  • Map Default Colors
  • Device Defaults
  • Network Defaults
  • Chart Defaults.
  • Device Thresholds.
  • NT Services & WMI

Note: You can also set preferences for a particular map using the Map Settings panel, available from the Edit menu. For more information, see Map Settings.
